# tategapy
This is a script to convert script to tategaki.

# Requirements
* Python (version >= 3)
It's only.


# Overview
Japanese is also possible to write vertical.
It call tategaki.
But normaly, computers always display horizonal.
So I make to convert script to tategaki.
